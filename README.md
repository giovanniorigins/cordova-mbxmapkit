cordova-mbxmapkit
=================

An Apache Cordova plugin for MapBox's MBXMapKit (for iOS only)

### Requirements

* Apache Cordova (3.5+)
* iOS (7.1+)

### Dependencies

* MapKit.framework
* libsqlite3.dylib
* [MBXMapKit](https://github.com/mapbox/mbxmapkit) (vendored)
